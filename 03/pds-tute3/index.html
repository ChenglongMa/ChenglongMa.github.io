Load dataset

We use pandas.read_csv() to load it as it’s a typical csv (comma-separated values) file. There are plentiful parameters in this method but only a few we will use commonly. To let pandas know we want to set its header , delimiter and the decimal point is “.”, we would do this:

  • sep specifies the delimiter of this file, which can accept regex, e.g., “[,;]“; Alternatively, you can use delimiter instead of sep.
  • decimal specifies the decimal point character.
  • header specifies row number(s) to use as the column names; NB: it can accept list of int to indicate multiple row headers. E.g., header=0 indicates the first line of data is the header; header=None indicates no header in file; header=[0,1] indicates the first two lines are the headers.
  • names specifies the columns names. “If the file contains a header row, then you should explicitly pass header=0 to override the column names.”

Some other parameters you may need:

  • skiprows: Line numbers to skip (0-indexed) or number of lines to skip (int) at the start of the file.
  • nrows: Number of rows of file to read. Useful for reading pieces of large files.
  • chunksize: Return TextFileReader object for iteration.
  • iterator: Return TextFileReader object for iteration or getting chunks with get_chunk().

For more detailed info, please refer to the pandas.read_csv doc.

Grouping statistics

.groupby() is a powerful method in pandas. You won’t feel strange if you have learned group by syntax in SQL. It splits the entire dataset based on unique values and then do some calculations or apply a function.

Let’s continue from sub_data and group it by Sex column:

sex_grp is a DataFrameGroupBy object which contains information of these groups and have multiple functions available.

  • .ngroups: return the number of groups
  • .groups: return all indices of groups
  • .mean(), .std(), .median(), etc. can provide corresponding stats information
  • .apply(), .aggregate(), etc. can accept callable functions to modify the groups

For more details, please check their documents respectively.
